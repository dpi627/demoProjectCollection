# 個人作品集網站開發任務清單

## 專案概述
基於 Bootstrap 5 + Font Awesome 的現代化個人作品集網站，專注展示 C# 開發專案。採用單頁 HTML 架構，具備完整的 CRUD 功能和響應式設計。

## 注意事項

1. **模組化開發**: 每個功能獨立開發，便於測試與維護
2. **漸進增強**: 確保基本功能在所有瀏覽器可用
3. **行動優先**: 優先考慮行動裝置使用體驗
4. **效能考量**: 保持首屏載入時間 < 3秒
5. **可擴展性**: 考慮未來 API 整合的架構設計

- **開發環境**: VS Code + Live Server 擴充
